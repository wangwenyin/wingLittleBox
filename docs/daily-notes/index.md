# 日常笔记

这里用于沉淀平时开发、阅读和工作中的零散记录。它不追求长文完整性，更强调及时归档与后续可检索。

## 建议的记录方式

- 每条笔记只记录一个主题
- 尽量写出触发场景和结论
- 如果有坑点，写清楚原因和排查路径
- 能附代码就附最小示例

## 当前分类建议

- 开发随记
- 阅读摘录
- 调试记录
- 工具配置
- 工作流优化

## 推荐模板

```md
## 标题

### 场景

### 现象

### 原因

### 解决方案

### 备注
```

后续可以继续按月份或主题拆分子页面。
