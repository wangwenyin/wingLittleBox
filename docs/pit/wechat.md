# 微信开发踩坑记录

微信环境下的问题通常带有很强的平台特性，排查时不要完全套用普通浏览器思路。

## 常见方向

### JSSDK 签名失败

排查顺序建议：

1. 当前 URL 是否正确参与签名
2. 后端签名是否使用了准确参数
3. 页面是否发生过 hash 路由变化
4. 域名是否完成平台配置

### 分享配置不生效

重点检查：

- 调用时机是否在 `ready` 之后
- 分享信息是否在页面切换后重新设置
- 是否命中了缓存或旧版本客户端问题

### WebView 兼容问题

常见现象：

- fixed 布局异常
- 输入框行为异常
- 某些 API 在 Android / iOS 表现不一致

## 建议

- 涉及微信能力时，前后端联调要尽早开始
- 记录平台版本、客户端版本和复现路径
- 对环境判断逻辑保持收敛，避免散落全项目
