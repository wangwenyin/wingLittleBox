# H5 踩坑记录

H5 的难点主要不在页面功能本身，而在移动端浏览器环境碎片化。

## 常见问题

### `100vh` 不稳定

移动端地址栏出现和收起时，`100vh` 可能变化，导致页面跳动。更稳妥的做法通常是结合动态视口单位或 JS 计算高度。

### 点击延迟与穿透

虽然现代浏览器大多已经改善，但在混合容器或旧环境里仍可能遇到点击事件异常。

### 输入框顶起布局

尤其在 iOS WebView 中，软键盘弹出会影响：

- 视口高度
- fixed 元素定位
- 页面滚动行为

### 图片与字体适配

需要关注：

- 高清屏资源清晰度
- 懒加载策略
- Web 字体加载时机

## 建议

- 真机验证比桌面模拟器更重要
- 与宿主容器相关的问题要单独记录
- 对关键页面建立一套最小兼容清单
