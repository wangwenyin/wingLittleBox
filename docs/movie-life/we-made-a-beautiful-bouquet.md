# 花束般的恋爱

> 拍的不是分手，而是同步率慢慢消失

这部电影很容易被概括成“相爱的人还是分开了”，但我觉得它真正拍到的，是两个人共享世界的方式一点点发生了偏移。

一开始，他们的靠近来自高度同步。喜欢的书、电影、音乐、散步路线，甚至吐槽的语气都能接上。那种关系最迷人的地方，不只是投缘，而是你感觉终于有人活在和你相似的感受频率里。

后来关系变淡，并不是因为谁突然变坏了，而是生活节奏开始改写两个人：

- 一个人被现实推着跑，开始优先考虑稳定与效率。
- 另一个人还想保留原来的感受密度，不愿意那么快妥协。
- 他们仍然理解对方，却已经不再同步体验同一个日常。

很多分开都不是戏剧性的。没有明确反派，没有惊天转折，只是共同语言慢慢减少，共同兴奋慢慢减少，连沉默都开始变得不一样。

所以这部片最难受的地方不是失去爱情，而是你亲眼看见一段关系从“我们正在一起长大”，变成“我们只能各自适应生活”。
