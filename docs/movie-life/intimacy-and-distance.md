# 亲密关系

> 最先消失的，常常不是爱，而是共享感受

我越来越在意电影里那些“不吵也不闹，但就是慢慢远了”的关系。

因为亲密关系真正变淡的时候，通常不是发生了某件足够明确的坏事，而是两个人逐渐不再共享同一种感受。你说的话对方仍然能听懂，但已经不会像以前那样自然地接住；你们依然关心彼此，却不再生活在同一个情绪节奏里。

这类电影让我反复想到一件事：关系的维持不只靠道德和责任，也靠持续更新彼此的理解。

所以我会把这个主题作为“电影人生”的常驻入口。后面写到相关作品时，我会重点观察：

- 两个人最初是靠什么靠近的。
- 他们的节奏是从哪里开始分叉的。
- 到最后，剩下的是遗憾、体面，还是一种平静接受。
