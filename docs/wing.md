# 关于 wing

这里作为个人简介与站点说明页使用。

## 我想记录什么

这个站点主要围绕几类内容持续积累：

- 前端学习与源码阅读
- 工具、环境与效率优化
- 开发踩坑与问题复盘
- 一些生活观察和长期兴趣

## 为什么叫“小盒子”

它更像一个长期可扩展的私人容器：不追求一次性完整，而是持续把零散知识、经验和想法收进去，慢慢形成自己的结构。

## 后续计划

- 持续补齐导航中各栏目内容
- 把零散笔记逐步沉淀成专题文章
- 增加更体系化的前端工程与源码分析内容
